# 🎨 Lovart Prompt Library（中文标准库）

本仓库用于系统化沉淀、管理和复用 Lovart 图像 / 视频生成提示词，
采用「反向拆解 + 模板化 + 版本化」方式构建可持续的创作资产。

## 📌 使用原则
- 全中文 Prompt
- 可直接复制到 Lovart 使用
- 明确正向 Prompt / 负面 Prompt
- 适合长期积累与复用

## 📂 目录结构
- remaster/      # 反向拆图 Prompt（从图片还原风格）
- styles/        # 风格类 Prompt（油画 / 水彩 / 插画）
- scenes/        # 场景类 Prompt
- subjects/      # 主体类 Prompt（动物 / 人物 / 工业）
- experiments/   # 实验中的 Prompt

## 🧠 Prompt 设计公式
主体 + 场景 + 构图  
情绪 + 风格 + 技法  
色彩 + 光影  
负面提示词（防翻车）
